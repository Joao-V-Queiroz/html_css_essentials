I Love Alpine


Crie um arquivo HTML em branco em algum lugar do seu computador com um nome como:i-love-alpine.html

Usando um editor de texto, preencha o arquivo com este conteúdo:


<html>
<head> <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script> </head>
<body> <h1 x-data="{ message: 'I ❤️ Alpine' }" x-text="message"></h1> </body>
</html>

Building a counter

Botão Contador

Vamos começar com um componente "contador" simples para demonstrar os fundamentos da escuta de estados e eventos no Alpine, dois recursos principais.


Insira o seguinte na <html> tag:


<div x-data="{ count: 0 }">
<button x-on:click="count++">Increment</button\>
<span x-text="count"></span>
</div>

Building a dropdown

Construindo um menu suspenso


Agora que vimos algumas funcionalidades básicas, vamos continuar e dar uma olhada em uma diretiva importante no Alpine: x-show, construindo um componente "suspenso" inventado.


Insira o seguinte código na <body> tag:


<div x-data="{ open: false }">
<button @click="open = ! open">Toggle</button>
<div x-show="open" @click.outside="open = false">Contents...</div>`
</div>

Building a search Input

Construindo um menu suspenso


Vamos agora construir um componente mais complexo e apresentar algumas outras diretivas e padrões.


Insira o seguinte código na <body> tag:


                        <div x-data="{
                            search: '',
                        
                            items: ['foo', 'bar', 'baz'],
                        
                            get filteredItems() {
                                return this.items.filter(
                                    i => i.startsWith(this.search)
                                )
                            }
                        }">
                        <input x-model="search" placeholder="Search...">

                        <ul>
                        <template x-for="item in filteredItems" :key="item">
                        <li x-text="item"></li>
                        </template>
                        </ul>
                        </div>
